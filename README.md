# Probability Simulation

Monte Carlo simulations for validating theoretical results in probability and statistics using Python.

## Overview

This project implements simulations for several classical probability problems, including waiting time analysis, order statistics, covariance analysis, Poisson splitting, and multinomial distributions. Each simulation compares empirical results with theoretical values to verify their correctness.

## Techniques

* Python
* NumPy
* Monte Carlo Simulation
* Random Variable Generation
* Statistical Analysis
* Probability Distributions

## Features

* Efficient vectorized implementation with NumPy
* One million simulation trials for accurate estimation
* Comparison between simulation and theoretical results
* Modular implementation with one problem per directory
